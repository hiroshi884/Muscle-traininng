## moneyとは
一言で言うと、家計簿管理サイトです。

### UVP（独自の価値提案 ≒ 一押しポイント）
簡単に自分の収入と支出がわかるサイト。

### 主なページ・機能
- Topページ
　　-登録フォーム
    -ログインフォーム

- 収入と支出の入力
    - 日付入力欄
    - メモ欄
    - 収入と支出入力欄
    -何に使ったかわかるようなイラスト入りカテゴリー選択
- 入力結果一覧ページ
　　　　-カレンダーのようになって収入と支出が見れる
- カテゴリー別でグラフ化されるページ
　　-円形グラフで月ごとの支出を表す
